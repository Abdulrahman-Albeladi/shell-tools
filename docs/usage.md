# Usage

This repository contains two Bash utilities in `bin/`.

## Running a script

From the repository root:

```bash
bash bin/sysmon.sh
bash bin/todo.sh
```

Or, after adding execute permissions:

```bash
./bin/sysmon.sh
./bin/todo.sh
```
