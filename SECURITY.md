# Security Policy

## Reporting

If you find a security issue in one of these scripts, report it privately to the repository owner through a non-public channel.

Include:

- affected script path
- operating system and shell version
- reproduction steps
- expected behavior
- actual behavior
- any relevant logs or terminal output with secrets removed

## Scope

This repository contains shell scripts. Likely risk areas include:

- unsafe handling of file paths
- command injection through untrusted input
- unsafe temporary-file handling
- accidental disclosure through logs or environment variables

## Disclosure

Please avoid opening a public issue for a suspected vulnerability until the owner has had time to review it.
